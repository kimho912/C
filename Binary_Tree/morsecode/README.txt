Files included: main.c, gen.c, gen.h, hw.c, hw.h, makefile

The main.c is to test the function in the hw.c.

Compilation instructions:

--Example 1 (running main.c):

./a.out (filename that has morse codes)

- ..- -..  -.-..# . ... .#

Don't forget to enter filename when you run it.
